Documentation

The ghanabills API is for finding and collecting the nation's bills of parliament in an organized manner for use in research and technical development.

Possible Queries

  • What bills did Hon. Gloria Akuffo publish in 2015?
  • What bills were covered the most from 2012-2017?
  • Which bill writers and topics receive the most attention from the media?
  • How many bills has Hon. Kwaku Ofori Asiamah published over his tenure?

Request Options

  • Author. E.g: Request bills written by Matthew Opoku Prempeh.
  • Bill Name. E.g: Request to see bills with the keyword "Legal".
  • Number of news mentions. E.g: Request bills mentioned in the news (Joy FM and Citi FM's pages) at least 10 times.
  • Year(s) published E.g: Request bills published from 2018-2020

Author

  • "author" is the key word used in the request
  • When typing the name(s), use underscores "_" instead of spaces.
  • Sample: /resources/bills?author=Ken_Ofori-Atta_Matthew_Opoku, which returns all bills written by Hon. Ken Ofori-Atta and Hon. Matthew Opoku Prempeh.

Bill Name

  • "name" is the key word used in the request
  • When typing the name(s), use underscores "_" instead of spaces.
  • Sample: /resources/bills?name=Legal_Profession_Bill, which returns the Legal Profession Bill and other bills with titles including the keywords "legal" and "profession".

Year(s) Published

  • "years" is the key word used in the request
  • Options: put one year (e.g. 2015), put a range (e.g. years=2010-2017), range from earliest, START-2015, range to latest 2004-END, all in database START-END
  • Multiple ranges can be attached (years=2012-2014_2017-2020 returns bills from 2012-2014 and 2017-2020), and single years and ranges can also be combined. (i.e. years=2014_2016-2020)
  • Sample: /resources/bills?years=START-2013, which returns the bills published from earliest date available through 2013.

Number of News Mentions

  • "news_mentions" is the key word used in the request
  • Options: put one number (e.g. news_mentions=10), put a range (e.g. 8-23), range from smallest (START-20), range to latest: 5-END, all in database START-END
  • Multiple Ranges can be attached (2012-2014_2017-2020 returns 2012-2014 and 2017-2020)
    • Multiple Ranges can be attached (5-8_20-END returns 5-8 and 20+)
  • Sample: /resources/bills?years=START-2013, which returns the bills published from earliest date available through 2013.

Combined Requests

  • "Combine the different search options with "&" symbol
  • Sample: /resources/bills?news_mentions=5-10_12-15&years=START-2019&name=legal&author=Gloria, which returns any bills with mentions between 5-10 or 12-15, written as early as possible through 2019, containing the word legal in the name, and an author named Gloria
  • To combine items within one search option (e.g. combine different author names), use the underscore symbol, i.e: author=John_Peter_Ambrose_Yaw_Osei-Adutum

To View all Bills

  • /resources/bills/all

Tips

  • Spell words correctly to ensure all related bills are retrieved.
  • Be specific; use as many of the 4 search options (author, bill name, news mentions, years) as possible to access a specific bill.
